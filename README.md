This mod is designed for version 1.21.1 with the NeoForge core.
The mod itself is about the Pocket Dimension; my mod works in both single-player and multiplayer.
To create your own world, you need to perform a ritual by finding, mining, or crafting 
1 Ender Pearl, 4 Emeralds, and 1 Lever. To enter the world, say  Prunus.
To exit the dimension, say Malus, but note that the first teleportation costs the ritual; subsequent ones cost 2 experience points and 4 hunger points. 

P.S. (Worlds are created 2,000 blocks apart, and each world is 4 chanks in size.) You can request to join a world / prequest. If you have a world, a friend must send you a request. You’ll receive a message with a bell icon; click “Accept.” If your friend has 2 EXP and enough hunger, voilà—you’re in the same world. Plus, if your friend misbehaves, you can kick them using the command /pkick
